# episode
A simple static blog generator    

效果见：[windfarer.github.io](http://windfarer.github.io/)

## Quick Start

* 新建项目

```
$ episode new <site_name>
```

* 生成页面

```
$ episode build
```

* 部署

```
$ episode deploy
```

* 启动本地服务器

```
$ episode server
```

* 监听项目变化并自动生成页面

```
$ episode watch
```
